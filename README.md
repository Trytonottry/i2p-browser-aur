# i2p-browser (AUR package)

This package provides the official I2P Browser Bundle for Linux.
It is a Firefox ESR build preconfigured to work inside the I2P network.

## Install

yay -S i2p-browser

## Launch

i2p-browser


## Files

- Installed to: `/opt/i2p-browser`
- Launcher: `/usr/bin/i2p-browser`
- Desktop entry: `/usr/share/applications/i2p-browser.desktop`
